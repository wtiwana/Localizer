Self-hosted setup

Pull model weights into the demo app, then load them from your own static hosting path. 

# Download the micro chat bundle (~78 MB)
npm run prepare:demo-self-hosted

# Start the demo in self-hosted mode
npm run dev:self-hosted

Models are served from /localizer-models/ and never fetched from Hugging Face at runtime.

Get started in a few lines

const ai = await Localizer.create({
  loadMicroAtStart: true,
});

for await (const chunk of ai.chat.stream('Hello!')) {
  console.log(chunk);
}

Install with npm install localizer or use the CDN — see the README.
